# Runbooks

Operational runbooks — step-by-step procedures for running and maintaining the system.

## Template

Each runbook should answer:

1. **When** to run this procedure (trigger / symptoms)
2. **Prerequisites** (access, tools, credentials)
3. **Steps** (numbered, copy-pasteable commands)
4. **Verification** (how to confirm success)
5. **Rollback** (how to undo)
